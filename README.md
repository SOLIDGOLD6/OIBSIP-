# Cybersecurity Internship Portfolio

## Overview

This repository contains the projects, research reports, practical labs, and security assessments completed as part of a Cybersecurity Internship Program offered by OASIS INFOBYTE. The internship focused on developing foundational, intermediate and advanced cybersecurity skills through hands-on technical exercises, security research, vulnerability assessments, and defensive security practices.

The objective of this internship was to gain practical experience with industry-standard cybersecurity tools, understand common security threats and vulnerabilities, and develop professional documentation and reporting skills.

---

## Internship Objectives

* Build practical cybersecurity knowledge through hands-on exercises.
* Develop experience using industry-standard security tools.
* Learn network and web application security fundamentals.
* Understand common cyber threats and attack techniques.
* Practice security analysis, reporting, and documentation.
* Strengthen defensive security and risk management skills.

## Internship Tasks

### Task 1 – Basic Network Scanning with Nmap

Perform network reconnaissance and service discovery using Nmap to identify open ports, running services, and operating system information within a controlled environment.

### Task 2 – Basic Firewall Configuration with UFW

Configure and manage firewall rules using UFW to control network traffic and implement basic host security controls.

### Task 3 – SQL Injection on DVWA

Explore SQL Injection vulnerabilities within a controlled lab environment using DVWA and document findings and remediation recommendations.

### Task 4 – Research Report: Common Network Security Threats

Research and analyze common network-based threats, their impact, attack methods, and recommended mitigation strategies.

### Task 5 – Research Report: Social Engineering Attacks

Investigate various social engineering techniques, real-world attack scenarios, and methods for preventing human-focused cyberattacks.

### Task 6 – Research Report: The Importance of Patch Management

Examine the role of patch management in cybersecurity, including vulnerability management processes and best practices.

### Task 7 – Vulnerability Scanning with Nikto

Perform web server vulnerability assessments using Nikto and analyze identified security findings.
